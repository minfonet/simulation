# Basic Multi-Agent Software System

## Where to create this structure

You must create this structure:

- inside your project root
- or inside a repository dedicated to the multi-agent system

Example:

```text
my-project/
├── agents/
├── .memory/
├── src/
├── backend/
└── frontend/
```

or:

```text
ai-org/
├── agents/
├── .memory/
├── runtime/
└── tools/
```

---

## Important rule

The folder:

```text
.memory/
```

must live in the operational root of the project.

Because it represents:

```text
organizational persistent memory
```

shared among all agents.

---

## What each folder does

### agents/

Defines agent identities and behavior.

---

### .memory/

Shared cognitive persistence.

---

### sessions/

Episodic continuity.

---

### active-context.md

Current operational state of the system.
