# Feature Catalog

## F001 — Realistic Environment Simulation

### Goal
Render realistic operational environments.

### Inputs
- map selection
- weather configuration
- traffic density
- obstacle presets

### Outputs
- live simulation environment

---

## F002 — Hardware Device Detection

### Goal
Automatically detect supported hardware devices.

### Supported Devices
- steering wheels
- pedals
- gear shifters
- joysticks
- custom USB devices

### Requirements
- dynamic driver registration
- calibration support
- persistent device profiles

---

## F003 — Training Session Management

### Goal
Create and manage simulation sessions.

### Requirements
- unique session ID
- organization-level isolation
- replay support
- telemetry capture

---

## F004 — Trainee Evaluation System

### Goal
Evaluate operational performance.

### Metrics
- reaction time
- operational precision
- safety compliance
- rule violations
- route completion
- collision count
- instructor scoring
